# 📋 학사일정 점검 시스템

학년도를 선택하면 공휴일이 자동 생성되며, 학사일정 엑셀 파일을 업로드하면 오류를 점검합니다.

## 주요 기능
- 🔄 **학년도 자동 전환**: 연도만 선택하면 해당 연도의 공휴일 자동 생성
- 📅 **음력 공휴일 자동 계산**: 설날·추석·부처님오신날 음력→양력 자동 변환
- ⚖️ **대체공휴일 자동 적용**: 토/일 겹침 시 대체공휴일 자동 계산
- ➕ **임시 공휴일 수동 추가/삭제**: 선거일 등 갑작스런 공휴일 대응
- 📂 **다중 파일 일괄 점검**: 여러 학교 동시 업로드 가능

## 점검 항목
1. 개학일 비교 (1~2학년 vs 3학년)
2. 1학기 수업일수 검증
3. 2학기 수업일수 검증

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```
